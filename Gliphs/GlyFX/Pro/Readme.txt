GLYPHSPRO


GlyphsPro is a collection of high colour 16x16 and 24x24 toolbar glyphs. They have been created in 24bit colour and saved in an optimised 256 colour pallete.

A number of the glyphs are unmodified (other than 
resizing and/or touching up) icons/toolbar glyphs taken from Windows XP and are copyright Microsoft however can be used in your applications.
